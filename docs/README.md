# JobPilot AI Documentation

Welcome to the documentation for the JobPilot AI project.

JobPilot AI is a cloud-native portfolio project being developed while studying for the AWS Certified Solutions Architect – Associate (SAA-C03) certification.

The goal is to build a production-inspired cloud application while documenting every architectural decision, engineering milestone, and AWS service introduced throughout development.

# Documentation Philosophy

Every document should answer one or more of the following questions:

- Why are we building this?
- What is the current state?
- How is it designed?
- What decisions were made?
- What remains to be completed?

Project documentation is treated as the authoritative source of truth for the current state of JobPilot AI.

---

# AI-Assisted Engineering

JobPilot AI follows an AI-assisted engineering workflow.

The primary AI tools are:

- **AWS JobPilot Mentor** — planning, AWS education, architecture, documentation, and project guidance.
- **Claude Code** — repository exploration, implementation assistance, debugging, testing, and code review support.

All AI-generated work is reviewed, tested, and approved by the developer before being committed.

# JobPilot AI Project Charter

## Project Vision

JobPilot AI is a cloud-native web application designed to help job seekers organize, track, and improve their job search.

The project is also a hands-on learning platform for mastering Amazon Web Services (AWS) while preparing for the AWS Certified Solutions Architect Associate (SAA-C03) certification.

Rather than simply studying AWS concepts, every service learned will be applied to a real-world software project.

---

# Objectives

- Earn the AWS Certified Solutions Architect Associate certification
- Build a professional cloud engineering portfolio
- Gain hands-on experience with AWS services
- Learn cloud architecture best practices
- Create a project suitable for technical interviews
- Document architectural decisions throughout development

---

# Target Users

Initially, the application is being built as a portfolio project.

Eventually, JobPilot AI may become a useful application for job seekers who want to:

- Track applications
- Upload resumes
- Store job descriptions
- Prepare for interviews
- Organize networking contacts
- Monitor application progress

# Success Criteria

The project will be considered successful if it:

- Demonstrates real AWS knowledge
- Uses modern cloud architecture

- Continues evolving as new AWS concepts are learned
